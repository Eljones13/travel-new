# TRAVEL RAVERS DESIGN SYSTEM

## The Vibe: Anime HUD / Survival OS
The app must look like a high-tech survival tool. 
- No white backgrounds. Use matte black `#0D0D0D`.
- Use "Glassmorphism" (frosted glass) for cards.
- Colors: Cyber Cyan (#00F2FF), Neon Magenta (#FF00FF), Safety Orange (#FF3E3E).

## UI Rules
- Borders: Ultra-thin (0.5px) with a neon glow.
- Typography: Monospaced fonts for all "Logistics" data (Lat/Long, Timestamps).
- Buttons: Gradient backgrounds with glowing pulse effects.
